# Анализ комплектации дорожной аптечки

Веб-приложение для автоматической проверки комплектации дорожной аптечки по фотографии с использованием нейросети YOLOv8.

## Как это работает

1. Пользователь загружает фото аптечки (или делает снимок с камеры на мобильном)
2. Нейросеть (YOLOv8) распознаёт предметы на изображении
3. Приложение сравнивает найденные предметы с обязательным списком
4. Выводится результат: комплектация полная или список недостающих предметов

## Что проверяется

| Предмет | Требуемое кол-во |
|---|---|
| Бинт большой | 3 |
| Бинт малый | 3 |
| Салфетки | 2 |
| Устройство для ИВЛ | 2 |
| Лейкопластырь | 1 |
| Инструкция | 1 |
| Карандаш | 1 |
| Термоодеяло | 1 |
| Медицинская маска | 1 |
| Перчатки | 1 |
| Ножницы | 1 |
| Блокнот | 1 |
| Жгут | 1 |

## Технологии

- **Flask** — веб-фреймворк
- **YOLOv8 (ultralytics)** — детекция объектов
- **OpenCV** — обработка изображений
- **NumPy** — численные вычисления

## Установка и запуск

### Локально

```bash
git clone https://github.com/imuzolev/medkit.git
cd medkit
python -m venv venv
```

Активация окружения:

```bash
# Windows
venv\Scripts\activate

# Linux / Mac
source venv/bin/activate
```

Установка зависимостей и запуск:

```bash
pip install -r requirements.txt
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5000`

### Docker

```bash
docker build -t medkit .
docker run -p 5000:5000 medkit
```
