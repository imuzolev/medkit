# Исправление ошибок webhook

## Проблема

Оба webhook delivery (push и ping) показывают красные иконки - это означает, что они не успешны.

## Что делать

### Шаг 1: Посмотреть детали ошибок

1. В GitHub откройте раздел "Recent Deliveries"
2. Кликните на каждый delivery (push и ping)
3. Посмотрите детали ошибки:
   - Какой статус код (302, 401, 404 и т.д.)?
   - Какое сообщение об ошибке?
   - Что в разделе "Response"?

### Шаг 2: Проверить URL webhook

Судя по предыдущим тестам, webhook возвращает 302 (редирект на login). Это означает:
- URL требует авторизации
- Или URL неправильный

### Шаг 3: Решения

#### Вариант 1: Использовать ручной деплой (рекомендуется)

Пока webhook не работает, используйте ручной деплой:

1. После каждого `git push`
2. Откройте Coolify UI: `http://2.56.240.119:8888`
3. Нажмите кнопку **"Deploy"** (справа вверху)
4. Coolify получит последний commit и задеплоит

**Это самый простой и надежный способ!**

#### Вариант 2: Удалить проблемный webhook и использовать только ручной деплой

1. В GitHub откройте: `https://github.com/imuzolev/medkit/settings/hooks`
2. Удалите проблемный webhook (который возвращает ошибки)
3. Используйте только ручной деплой через кнопку "Deploy" в Coolify

#### Вариант 3: Попробовать другой формат URL

Если хотите попробовать исправить webhook, попробуйте:

1. Удалить текущий webhook в GitHub
2. В Coolify UI откройте раздел "Webhooks"
3. Проверьте, есть ли там другой URL (не с `/manual`)
4. Если есть, добавьте его в GitHub

Но скорее всего, это не сработает, так как проблема в авторизации.

## Рекомендация

**Используйте ручной деплой** - это самый простой и надежный способ:

✅ Работает всегда  
✅ Не требует сложной настройки  
✅ Вы контролируете процесс  
✅ Занимает всего несколько секунд  

Просто нажимайте кнопку "Deploy" в Coolify UI после каждого `git push`.

## Проверка деталей ошибок

Чтобы понять точную причину, откройте каждый delivery в GitHub и посмотрите:
- Статус код
- Сообщение об ошибке
- Response body

Это поможет понять, можно ли исправить webhook или лучше использовать ручной деплой.
